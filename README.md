# InshortApi

This will fetch the news from https://inshorts.com/

## Requirements

1. Requests module
2. BeautifulSoup 

```bash
pip install requests bs4 
```


## USAGE

```bash
git clone https://github.com/xaviruvpadhiyar98/InshortApi.git
cd InshortAPI/
```

```python3
from inshortapi import InshortAPI, ShowCategories
ShowCategories()
InshortAPI()
```
